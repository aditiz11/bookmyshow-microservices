# BookMyShow Microservices

A distributed movie ticket booking platform built using Spring Boot microservices.

## Architecture

Client
↓
API Gateway
↓
User Service
Movie Service
Booking Service

## Features

* JWT Authentication
* Spring Security
* API Gateway
* RESTful APIs
* MySQL Database
* Microservices Architecture

## Tech Stack

* Java 21
* Spring Boot 3.5
* Spring Security
* Spring Cloud Gateway
* Maven
* MySQL

## Services

### User Service

* User Registration
* User Login
* JWT Token Generation

### Movie Service

* Movie CRUD Operations
* Movie Listing APIs

### Booking Service

* Ticket Booking APIs

### API Gateway

* Centralized Routing
* Request Forwarding
* Service Isolation
